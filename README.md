# Inverted Pendulum

## Simulation of inverted pendulum with PID controller (adjustable PID paramters)

![pendulum_start](./pendulum/pics/Pendulum.gif)

intsall:
- pygame
- box2d
- box2d-py

run the testbench

keyboard commands:
- d: delete world
- c: create world
- m: manual mode
- a: automatic mode


uses PID library from [PyConSys](https://github.com/mck-sbs/PyConSys)
